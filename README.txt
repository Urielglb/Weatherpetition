Para la correcta ejecución del programa lo que se debe hacer es correr desde terminal el archivo main.py con python3 poniendo como segundo argumento la direccion en donde se encuentra el archivo con los vuelos
Ejemplo de ejecucion $ python3 main.py Direccion/del/archivo
Para que funcione el programa el archivo debe tener la siguiente estructura:
Pais de origen Pais de destino Latitud de origen Longitud de origen Latitud de destino Longitud de destino 
Con solo los datos de un viaje por linea en ese orden estricto, el programa empezará a leer desde la segund linea, pues asume  que en la primera estan las descripciones del documento
